# Disaggregated computing related talks

## About this book

This document lists links to YouTube content and transcript texts for 244 disaggregated computing-related talks published on YouTube between March 2017 and May 2024, by publication date and speaker (organisation and name of affiliation).  
This publication does not include titles of speeches or presentation materials.

Please note that although the transcription has been prepared with the utmost care and accuracy, it may contain mishearing or typographical errors.

Each entry consists of the speaker's name, a link to the YouTube content ("YouTube") and a link to the transcribed text ("Text").  
In the case of talks by several speakers, the same link is duplicated for each speaker.
